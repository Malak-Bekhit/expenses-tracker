 Expense Tracker (Python)

A simple Python project that helps users track daily expenses, categorize spending, and generate reports.

---

# Features
- Add new expenses (amount, category, date)
- Store data in CSV file
- View total spending
- Show top spending category

---

# Technologies Used
- Python
- CSV module
- defaultdict (for data aggregation)
